# Changelog

## v1.1.1 - 2026-06-25
- 펭귄 러너를 7프레임 이미지 기반 모션으로 교체.
- 러닝 프레임 속도, 좌우 체중 이동, 몸통 회전, 그림자 변화를 보정.
- 발 디딤 순간의 작은 눈발 효과를 추가해 달리는 느낌을 강화.
- 현재 상태를 `release/v1.1.1` 스냅샷으로 저장.

## v1.1 - 2026-06-25
- 5레인 원근형 세로 러너를 무한 진행 방식으로 전환.
- 시작 화면 단순화 및 `3, 2, 1, GO!` 카운트다운 적용.
- 단일 펭귄 러너 캐릭터와 2프레임 러닝 애니메이션 적용.
- 적 캐릭터, 독극물 투사체, 수집 코인/캐릭터 시각 리소스 적용.
- 단계 상승마다 속도와 생존 게이지 감소 속도 증가.
- `스피드 업!!` 알림 위치 조정.
- 동전/장애물/적군/투사체 판정을 실제 화면 충돌 박스 기준으로 개선.
- pause/play 버튼, 생존 게이지, 거리/단계 HUD, 결과 순위 화면 적용.
- 도로 5칸 폭과 오브젝트 중앙 정렬 계산 통일.
- 현재 상태를 `release/v1.1` 스냅샷으로 저장.

## v0.1.0-demo - 2026-05-28
- 3스테이지 러너 웹 데모 구현 (60~90초 플레이 길이).
- 캐릭터 3종 및 각 캐릭터 액티브 스킬 구현.
- 실패 즉시 재도전, 결과 공유 버튼, A/B 위치 실험 구현.
- Telemetry 이벤트(`session_start`, `stage_start`, `hit_obstacle`, `stage_clear`, `run_end`, `retry_click`, `share_click`) 저장 구현.
- CharacterManifest, StageConfig, TelemetryEvent 계약 검증 코드 및 테스트 추가.
- 1주 데모 실행/상용화 확장 로드맵 문서 추가.
