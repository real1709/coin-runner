# 상용화 확장 로드맵 (데모 이후 8~10주)

## Phase 1 (주 1~2): KPI 진단
- 데모 데이터 분석: 완주율, 평균 플레이타임, 재시도율, 공유 클릭률
- 유지/개선 포인트 확정 및 우선순위 선정

## Phase 2 (주 3~5): 콘텐츠 확장
- 스테이지 10+ 구성
- 캐릭터 10+ 확장
- 시즌 이벤트 기본 구조 도입

## Phase 3 (주 6~7): 라이브 기능
- 출석/랭킹/기간 미션 추가
- 이벤트 보상 테이블 운영 정책 수립

## Phase 4 (주 8~10): 모바일 상용 배포
- Android 우선 배포 파이프라인 구축
- iOS 후속 배포 준비
- Unity 재구축 여부 최종 결정 (KPI 기준)
