# 테스트 및 KPI 계획

## 기능 테스트
- 점프/회피/스킬 입력이 모바일/키보드에서 정상 동작
- 장애물 충돌 시 런 종료 및 결과 화면 전환
- 재도전 버튼 클릭 시 즉시 새 세션 시작
- 스테이지 전환(1->2->3)과 클리어 연출 정상 동작

## 밸런스 테스트
- 초심자 5명 기준, 첫 3회 플레이 내 최소 1회 Stage 1 클리어
- Stage 2 진입률 50% 이상, Stage 3 진입률 25% 이상 목표

## 성능 테스트
- 모바일 브라우저 기준 50~60fps 목표
- 초기 로딩 5초 이내 목표

## IP 검수 테스트
- 캐릭터 외형/대사/연출 체크리스트 기준 주 1회 승인
- 수정 요청 반영 후 재승인 리드타임 기록

## 마케팅 KPI
- 첫 세션 완료율 60%+
- 평균 플레이 시간 3분+
- 재시도율 35%+
- 공유 클릭률 5%+

## Telemetry 이벤트 점검
- 필수 이벤트 누락 여부 확인:
  - `session_start`
  - `stage_start`
  - `hit_obstacle`
  - `stage_clear`
  - `run_end`
  - `retry_click`
  - `share_click`
