# Changelog

## v1.1.4 - 2026-07-01
- 결과 순위를 전체 표시 + 스크롤 방식으로 개선하고, 현재 플레이 기록 위치로 자동 스크롤되도록 조정.
- `ice_crack`, `ice_wall` 장애물을 이미지 기반으로 교체하고 `small_block` 장애물 스폰을 제거.
- 장애물별 크기 제어 옵션(`laneWidthRatio`, `sizeScale`, `heightScale`, `groundOffsetRatio`)을 추가.
- 결과창에 `도발 한마디` 입력/저장 UI를 추가하고, 순위 항목 오른쪽에 문구와 작성자 닉네임을 표시.
- 저장 후 도발 문구 입력칸이 자동으로 비워지도록 동작 변경.
- 로컬 저장 랭킹 외에 서버 공용 랭킹 API(`/api/leaderboard`) 저장/조회를 추가해 링크 사용자 간 순위 공유 지원.
- 현재 상태를 `release/v1.1.4` 스냅샷으로 저장.

## v1.1.3 - 2026-06-30
- 타이틀 화면을 캐릭터 포함 이미지와 신규 `Tap to Play` 이미지로 교체.
- 결과 화면을 타이틀과 맞는 그림체의 이미지 기반 화면으로 교체.
- 타이틀 BGM, 시작 클릭 효과음, 카운트다운 효과음, GO 효과음, 인게임 루프 BGM을 추가.
- 인게임 배경 이미지를 적용하고 코인/장애물이 도로 시작 부근에서 등장하도록 조정.
- 좌우 배경 장식 오브젝트(나무/표지판/이글루)를 추가.
- 좌우 장식 오브젝트 수를 3개로 줄이고, 크기/간격/도로 평행 위치를 보정.
- 두더지 장애물은 상반신 중심으로 보이도록 표시 영역을 조정.
- 현재 상태를 `release/v1.1.3` 스냅샷으로 저장.

## v1.1.2 - 2026-06-26
- 무한 부스터 아이템을 추가하고, 사용 후 20초 동안 재등장하지 않도록 조정.
- 근육 두더지 장애물을 추가하고 `slope_hole`을 두 칸 폭 장애물로 변경.
- 두더지는 3단계부터 계속 랜덤 등장하되, 단계가 올라갈수록 등장 빈도가 서서히 감소하도록 조정.
- 응가를 던지는 적군 캐릭터 크기를 칸 폭에 더 맞게 확대.
- 5단계 이상 장애물에서 `storm_gate`를 제거.
- `moving_crate`를 이미지 기반 이동 적군으로 교체하고 좌/우 한 칸 랜덤 이동 패턴을 추가.
- 자석 아이템은 제거하고 부스터 중심 아이템 구조로 정리.
- 현재 상태를 `release/v1.1.2` 스냅샷으로 저장.

## v1.1 - 2026-06-25
- 5레인 원근형 세로 러너를 무한 진행 방식으로 전환.
- 시작 화면 단순화 및 `3, 2, 1, GO!` 카운트다운 적용.
- 단일 펭귄 러너 캐릭터와 2프레임 러닝 애니메이션 적용.
- 적 캐릭터, 독극물 투사체, 수집 코인/캐릭터 시각 리소스 적용.
- 단계 상승마다 속도와 생존 게이지 감소 속도 증가.
- `스피드 업!!` 알림 위치 조정.
- 동전/장애물/적군/투사체 판정을 실제 화면 충돌 박스 기준으로 개선.
- pause/play 버튼, 생존 게이지, 거리/단계 HUD, 결과 순위 화면 적용.
- 도로 5칸 폭과 오브젝트 중앙 정렬 계산 통일.
- 현재 상태를 `release/v1.1` 스냅샷으로 저장.
